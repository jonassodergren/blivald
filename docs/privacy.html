1. Blivald.se

Privacy

Vårat mål är att spara minimalt med information om dig

Intro

Denna policy beskriver hur blivald.se hanterar personlig data om användarna. Dokumentet uppdaterades den 2020-05-20.

Avsändare

Sajten blivald.se ägs och drivs av aktiebolaget Softwhere AB. Företagets organisationsnummer är 559002-1811, och den huvudsakliga verksamheten är dataprogrammering.

Cookies eller kakor

Inga kakor används i dagsläget.

E-post och konto

Tjänsten skickar inte några reklamutskick till den person som registrerat sin e-postadress på blivald.se. För att tjänsten skall kunna veta om en användare har skapat ett certifikat innan så skapas automatiskt ett konto(e-postadressen som nyckel) för den som utfärdar ett certifikat. Varje konto får ett unikt ID-nummer knytet till sig som används i loggning och för att identifiera certifikatet.

Loggning

Varje gång du besöker en sida på internet, skickas en hel del information om dig till en server. Servern som driver blivald.se loggar information när du utfärdar eller verifierar ett certifkat. Informationen som loggas knyts inte till dig som individ(din e-postadress) utan till ett unikt ID-nummer som står på ditt certifikat. Anledningen är att blivald.se vill göra det omöjligt att identifera en användare genom att läsa loggfilen. Information om vad som sker(skapa eller utfärda) loggas och vid vilken tidpunkt det sker.

Spridning av din personliga data

Blivald.se säljer ingen data som samlas in, exemplevis din e-postadress.

Informationssäkerhet

Vi ämnar att hålla sajten och din information så säker som möjligt. Men eftersom vi inte kan garantera det så försösker vi istället designa tjänsten på ett sätt så att vi inte behöver lagra information om dig. Exempelvis använder vi tjänsten Auth0 för att du ska kunna logga in säkert. Alla av oss kända fel eller sabotage mot tjänsten kommer att delges dig så fort vi får kännedom om de.

GDPR

Kontakta oss för att få ett fullständigt utrag om vilken information blivald.se har om dig. Blivald.se trovärdighet mot företagare och jobbsökare bygger på att en individ inte kan utfärda flera certifikat. För att undvika att någon skapar ett certifikat, ber om att bli fullständigt raderad, och sedan registrerar sig igen för att utfärda ett certifikat har vi designat en unik lösning för att ta bort all information om dig samtidigt som vi kan avgöra om du har vasrit kund innan.

Ställ en fråga till info@blivald.se med ämnesraden:

  • “GDPR - Rätt att bli glömd” - så tar vi bort all information om dig.
  • “GDPR - Informationsutdrag” - så skickar vi en fil med all information som vi har om dig.

Kort om blivald.se

Hjälpa de som söker arbete och de som söker arbetskraft. Certifikatet är unikt och möjliggör för jobbsökaren att förmedla vilket jobb/företag/arbetsplats den helst föredrar.
