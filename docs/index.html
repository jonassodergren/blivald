Frågor

Vad betyder certifikatet?

Att en arbetsgivare har blivit vald. Enkelt att förklara och lita på. En individ får bara skapa ett certifikat.

Behövs CV?

Certifikatet är ett komplement till dina befintliga ansökningshandlingar. Ersätter inga befintliga dokument som CV.

Hur skickar jag certifikatet?

Maila gärna arbetsgivaren direkt. Om du ansöker via ett system kan certifikatet laddas upp i fältet där CV:n efterfrågas.

Kort om blivald.se

Vill hjälpa individer att intyga sin vilja. Vilken arbetsgivare vill inte träffa en individ som valt de bland över en miljon företag i Sverige.
